# Backend API

이 디렉토리는 API 계층의 개념적 위치를 설명합니다. 실제 Route Handlers는 현재 [frontend/app/api](../../frontend/app/api) 아래에 둡니다.

## 범위

- 업로드 URL 발급
- 자료/녹화본 상태 조회
- QA/퀴즈 요청 처리

## 계약 문서

- API 초안: `docs/openapi.yaml`
- DB 스키마: `docs/schema.md`
- 프론트/데모 흐름: `docs/user-flow.md`

## 현재 상태

- Next.js Route Handlers + Prisma 기반 구현을 시작하는 단계입니다.
